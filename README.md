# Quantum Teleportation with Qiskit

This project implements the standard quantum teleportation protocol using Qiskit.

## What it does
- Builds a 3-qubit teleportation circuit
- Simulates the protocol using Qiskit's Aer simulator
- Verifies that the final state of the target qubit matches the initial state

## Tech stack
- Python
- Qiskit
- Jupyter Notebook

## How to run
```bash
pip install -r requirements.txt
jupyter notebook quantum_teleportation.ipynb
